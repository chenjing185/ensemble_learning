报错信息:

 
xxxxxxxxxx
Uncaught TypeError: Cannot read property 'value' of null
    at 08.获取元素.html:7

以上报错是因为:程序是从上往下执行,html->body还没加载,所以找不到id叫作num的对象

解决办法1:将js脚本 放body后面执行

解决方法2:通过window.onload=function(){...}

 
xxxxxxxxxx
<script>
            //alert(document.getElementById("num").value);
            //body的所有标签加载完后才执行函数中的内容
            window.onload=function  () {
                alert(document.getElementById("num").value);
            }
        </script>
    </head>

2.js的正则表达式

RegExp:是正则表达式regular expression的简写

参考w3school的 JavaScript RegExp 参考手册。

方括号—用于查找某个范围内的字符
表达式描述
[abc]查找方括号之间的任何字符。
[^abc]查找任何不在方括号之间的字符。
[0-9]查找任何从 0 至 9 的数字。
[a-z]查找任何从小写 a 到小写 z 的字符。
[A-Z]查找任何从大写 A 到大写 Z 的字符。
[A-z][a-zA-Z]查找任何从大写 A 到小写 z 的字符。
[adgk]查找给定集合内的任何字符。
[^adgk]查找给定集合外的任何字符。
(red|blue|green)查找任何指定的选项。
元字符—拥有特殊含义的字符
元字符描述
.查找单个字符，除了换行和行结束符。
\w查找单词字符。单词字符包括：a-z、A-Z、0-9，以及下划线。
\W查找非单词字符。
\d [0-9]查找数字。
量词
量词描述
n+匹配任何包含至少一个 n 的字符串。
n*匹配任何包含零个或多个 n 的字符串。
n?匹配任何包含零个或一个 n 的字符串。
n{X} 6{3}要有3个6匹配包含 X 个 n 的序列的字符串。
n{X,Y}7{1,5}匹配包含 X 至 Y 个 n 的序列的字符串。
n{X,}匹配包含至少 X 个 n 的序列的字符串。
n匹配任何结尾为 n 的字符串。
^n ^[0-9]匹配任何开头为 n 的字符串。
?=n匹配任何其后紧接指定字符串 n 的字符串。
?!n匹配任何其后没有紧接指定字符串 n 的字符串。
RegExp 对象的方法
方法描述
test检索字符串中指定的值。返回 true 或 false。
