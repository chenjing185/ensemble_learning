一.jQuery简介

jQuery，顾名思义，也就是JavaScript和查询（Query），即是辅助JavaScript开发的库,

由美国人John Resig于2006年创建，吸引了来自世界各地的众多JavaScript高手加入。 如今，jQuery已经成为最流行的JavaScript库，在世界前10000个访问最多的网站中，有超过55%在使用jQuery。 jQuery是一个兼容多浏览器的JavaScript库，核心理念是:写得少,做得多

三.jQuery语法

 
​x
jQuery 语法是通过选取 HTML 元素，并对选取的元素执行某些操作。
​
基础语法： $(selector).action()
​
美元符号定义 jQuery
选择符（selector）"查询"和"查找" HTML 元素
jQuery 的 action() 执行对元素的操作
实例:
​
$("p").hide() - 隐藏所有 <p> 元素

五.不同对象写法

1.dom对象转成jq对象 
  语法:把js对象用$()包起来:    $(dom对象)  $(uname).val()
  
2.jq对象转成dom对象:
  语法:jq对象[0]  uname2[0]  get(0)
 
xxxxxxxxxx
$(function  () {
                //alert("简写方式:jq加载");
                var rs=document.getElementById("p1");
                var result=rs.innerHTML;
                //转成jq对象  $(js对象)
                var result2=$(rs).html();
                console.log("js2方式获取:"+result2);
                
                //jq方式
                var $rs2=$("#p1");
                var result3=$rs2.html();
                //转成js对象  jq.get(0)  jq[0]
                //var result4=$rs2.get(0).innerHTML;
                var result4=$rs2[0].innerHTML;
                console.log("jq2方式获取:"+result4);
            })
        </script>

六.选择器

 
x
    function show1 () {
                //按id进行选择,获取值
                alert($("#i1").val());
            }
            
            function show2 () {
                //按标签进行选择,有入参
                //使用eq(index):获取第N个元素,index从0开始
                //$("input").eq(1).val("新的值");
                //eq选择器
                $("input:eq(1)").val("新的值");
            }
            
            function show3 () {
                //按class进行选择,有入参
                $(".myclass").val("新的值2");
            }
            
            function show4 () {
                //按name属性进行选择,有入参
                $("[name='sty']").val("新的值3");
            }
        </script>
    </head>
    <body>
        <input type="text" id="i1" value="123" /><br/>
        <input type="text"  value="" /><br/>
        <input type="text" class="myclass" value="" /><br/>
        <input type="text" name="sty" value="" /><br/>
        
        <input type="button"  value="按id显示value" onclick="show1()"/><br/>
        <input type="button"  value="按标签名修改value" onclick="show2()"/><br/>
        <input type="button"  value="按class修改value" onclick="show3()"/><br/>
        <input type="button"  value="按属性修改value" onclick="show4()"/><br/>
    </body>
